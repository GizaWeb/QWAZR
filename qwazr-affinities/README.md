QWAZR Affinities
===========================

A recommendation engine for [QWAZR](https://www.qwazr.com). 

Requirement
-----------

- Java Runtime 7
- OpenSearchServer v1.5.10 or newer
- Maven 3.0 or newer (for building only)

Usage
-----

The provided [REST/Json Web Service is described here](src/doc/usage.md).
